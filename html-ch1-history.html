บทที่ 1 HTML คืออะไร & ประวัติความเป็นมาอย่างละเอียด

🕰️ History (ประวัติความเป็นมา)

HTML (HyperText Markup Language) เริ่มต้นคิดค้นขึ้นโดย Tim Berners-Lee นักวิทยาศาสตร์คอมพิวเตอร์แห่งสถาบัน CERN ในปี 1991 โดยเขาต้องการระบบแบ่งปันข้อมูลเอกสารวิจัยที่สามารถกดลิงก์เชื่อมโยงไปยังหน้าอื่นๆ ได้ผ่านเครือข่ายอินเทอร์เน็ต

📦 ยุคเปลี่ยนผ่าน: HTML4 VS XHTML

HTML4 (1997): ประสบความสำเร็จมาก แต่โครงสร้างไวยากรณ์หละหลวมเกินไป เช่น ไม่ยอมปิดแท็ก หรือพิมพ์พิมพ์ใหญ่พิมพ์เล็กสลับกันเบราว์เซอร์ก็ยังยอมให้อภัยทำงานต่อได้

XHTML (2000): เพื่อแก้ไขความหละหลวม องค์กรสากลจึงพยายามจับ HTML มาแต่งงานกับ XML ทำให้เกิดไวยากรณ์ที่เข้มงวดสูงมาก เช่น ทุกแท็กต้องเขียนด้วยตัวพิมพ์เล็กเสมอ ต้องปิดแท็กตัวเองด้วยสแลช (เช่น <br />) เสมอ หากเขียนผิดแม้แต่จุดเดียว เบราว์เซอร์จะหยุดวาดหน้าเว็บทันที ส่งผลให้นักพัฒนาทำเว็บได้ยากขึ้น

🚀 ปฏิวัติวงการสู่ HTML5

HTML5 (2014): เกิดขึ้นมาเพื่อทลายความเครียดของ XHTML และตอบโจทย์เว็บยุคใหม่ที่ไม่ได้มีเพียงแค่ข้อความและรูปภาพ แต่ต้องการระบบเล่นเสียงและวิดีโอแบบไม่ต้องลงปลั๊กอินเสริม (Native Multimedia Support) รวมถึงการกำหนดความหมายโครงสร้างข้อมูลแบบเป็นธรรมชาติต่อหุ่นยนต์เสิร์ชเอนจิน

🏢 W3C VS WHATWG (ศึกผู้กำหนดมาตรฐาน)

W3C (World Wide Web Consortium): นำโดย Tim Berners-Lee เน้นไปทางวิชาการและอยากผลักดัน XHTML ให้เสร็จสมบูรณ์ แต่ผู้พัฒนาเว็บส่วนใหญ่และผู้สร้างเบราว์เซอร์มองว่าไม่สอดคล้องกับพฤติกรรมจริงในตลาด

WHATWG: เกิดจากการรวมตัวกันของค่ายผู้ผลิตเบราว์เซอร์ยักษ์ใหญ่ (Apple, Mozilla, Opera) ในปี 2004 เพื่อพัฒนาเทคโนโลยีเว็บที่ใช้งานได้จริงในชีวิตประจำวัน จนในที่สุดปี 2019 ทั้งคู่ได้ข้อตกลงร่วมกันว่า **WHATWG จะเป็นผู้กำหนดมาตรฐานหลักของเอกสาร HTML (เรียกว่า Living Standard)** ซึ่งหมายถึงจะไม่มีการเปลี่ยนเลขเวอร์ชันเป็น HTML6 หรือ HTML7 อีกต่อไป แต่ตัวภาษาจะอัปเดตฟังก์ชันใหม่ๆ เข้าไปในระบบอย่างต่อเนื่องตลอดเวลา
