บทที่ 20-22 Event Loop, Prototype & Memory Management (Senior Level)

Event Loop: กลไกคุมความต่อเนื่องแบบ Single Thread ทำการแยกระหว่าง Call Stack และแถวพักข้อมูลรอรอบการทำงานอย่าง Microtask (เช่น Promise) และ Macrotask (เช่น setTimeout)

Memory Architecture: การแยกเก็บข้อมูลตัวแปรธรรมดาใน Stack และวัตถุขนาดใหญ่ซับซ้อนใน Heap พร้อมระบบเคลียร์ขยะอัตโนมัติ (Garbage Collection)

ภาค 4 : สิ่งที่ Developer ต้องเรียนรู้ต่อหลังจบ HTML/CSS/JS

1. Git / Package Managers: ฝึกฝนการใช้ Git (commit, branch) และจัดการ Dependencies ผ่าน npm/pnpm

2. TypeScript: การต่อยอดมาใช้ระบุไทป์และอินเทอร์เฟส ครอบทับ JS เพื่อลดบ๊กในระบบงานใหญ่

3. Frameworks: ก้าวสู่งานสเกลระดับ Enterprise ด้วย React, Next.js (SSR/SSG), Node.js สำหรับทำ API และฐานข้อมูล SQL/NoSQL

4. DevOps & Cloud: ปิดท้ายด้วยการทำ Containerization ด้วย Docker และผูกท่อ CI/CD สู่ระบบ Cloud
