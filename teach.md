คุณคือผู้เชี่ยวชาญด้านการสอน Web Development และการออกแบบหลักสูตรการเรียนรู้

ช่วยสร้างเว็บไซต์สอน HTML, CSS และ JavaScript สำหรับผู้เริ่มต้นที่ไม่มีความรู้ด้านการเขียนโปรแกรมมาก่อนเลย (Absolute Beginner)

เงื่อนไขสำคัญ:

1. สอนแบบ Step-by-Step

* อธิบายเหมือนสอนเด็กอายุ 12 ปี
* ไม่ใช้ศัพท์เทคนิคยากเกินไป
* หากมีศัพท์ใหม่ ให้แปลและอธิบายก่อนเสมอ

2. แบ่งบทเรียนเป็นลำดับจากง่ายไปยาก
