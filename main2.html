บทที่ 1 HTML คืออะไร & ประวัติความเป็นมา

HyperText Markup Language คือภาษาโครงสร้างหลักของเว็บไซต์ทั่วโลก

History: จุดเริ่มต้นจาก Tim Berners-Lee ปี 1991
HTML4 & XHTML: ยุคแห่งความเข้มงวดของ Syntax (XML-based)
HTML5: ยุคปัจจุบัน รองรับ Multimedia และ Semantic Web
W3C & WHATWG: สององค์กรผู้ร่วมกำหนดมาตรฐานเว็บในปัจจุบัน
index.html (ทดลองแก้ไขโค้ดด้านล่างได้)

บทที่ 2 HTML Document Structure

โครงสร้างพื้นฐานที่ทุกๆ ไฟล์ HTML จำเป็นต้องมีเพื่อให้เบราว์เซอร์อ่านค่าได้ถูกต้อง

<!DOCTYPE html>: ประกาศว่านี่คือ HTML5
<html>: รากเหง้า (Root Element) ของเอกสาร
<head>: ส่วนเก็บข้อมูล Metadata, Title, Link CSS (ไม่แสดงบนเว็บ)
<body>: ส่วนเนื้อหาหลักที่จะไปปรากฏให้ผู้ใช้งานเห็น

บทที่ 12 Semantic HTML

บทที่ 20 Event Loop & Concurrency Model (Senior Level)

เจาะลึกกลไกเบื้องหลังของ V8 Engine ในการทำงานแบบ Single-threaded แบบ Asynchronous ไม่ให้เว็บค้าง

Call Stack: ด่านหน้าทำคำสั่ง Synchronous ทันทีแบบ LIFO
Web APIs: ขอยืมแรงเบราว์เซอร์ทำของช้า เช่น setTimeout, fetch
Microtask Queue: แถวแซงคิวพิเศษระดับสูง (Promise.then, async/await)
Callback Queue: แถวปกติสำหรับการทำงานระดับแมคโคร (setTimeout)
"Event Loop มีหน้าที่คอยตรวจสอบว่าถ้า Call Stack ว่างลง มันจะวิ่งไปหยิบงานจาก Microtask Queue มาทำก่อน แล้วค่อยตามด้วย Callback Queue เสมอ"

ภาค 4 : สิ่งที่ Developer ต้องเรียนต่อในโลกปัจจุบัน

หลังจากแม่นยำ HTML/CSS/JS แล้ว นี่คือสแต็กเทคโนโลยีที่คุณต้องเลือกเดินต่อไปเพื่อก้าวสู่สายอาชีพแบบมืออาชีพ

ประเภทเครื่องมือ เทคโนโลยียอดฮิต ความสำคัญ
Version Control Git (GitHub / GitLab) บันทึกประวัติการแก้โค้ด ทำงานร่วมกันเป็นทีมโดยโค้ดไม่พังทับกัน
Build Tool & Package npm / pnpm / Vite จัดการดาวน์โหลด Lib ภายนอก และรวมไฟล์มัดแพ็กเว็บให้เร็วปรี๊ด
Frontend Framework TypeScript + React / Next.js ทำแอปพลิเคชันขนาดใหญ่ด้วย Component และดึงข้อมูลแบบ SSR / SSG
Backend & Database Node.js (Express) + PostgreSQL / MongoDB สร้าง API, จัดการระบบล็อกอิน (JWT), เก็บฐานข้อมูลถาวรอย่างปลอดภัย
Deployment & Cloud Docker / Vercel / AWS / CI-CD จำลอง Container เสกเว็บขึ้นสู่อินเทอร์เน็ตจริงให้ผู้ใช้งานทั่วโลกเข้าชม
