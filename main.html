Variable Types Function Control

JavaScript: ตัวแปร ชนิดข้อมูล และโครงสร้างควบคุม

  • Type & ตัวแปร: รู้จัก var let const, string, number, boolean, array, object
  • Function: การสร้างและเรียกใช้ฟังก์ชัน
  • Control Structure: if/else, while, do..while, for

ชนิดตัวแปรใน JavaScript

ตัวแปร (Variable)
  • ประกาศด้วย var, let, const
  • ชนิดข้อมูลหลัก:
    • String - ตัวอักษร: let name = "นารี";
    • Number - ตัวเลข: let age = 15;
    • Boolean - จริง/เท็จ: let active = true;
    • Array - กลุ่มข้อมูล: let scores = [10, 15, 20];
    • Object - เก็บข้อมูลซ้อน (key/value): let person = {name:"นารี", age:15};
var (เก่า), let (ตัวแปรเปลี่ยนค่าได้), const (ค่าคงที่ เปลี่ยนค่าไม่ได้)
ตัวอย่างการสร้างตัวแปรชนิดต่าง ๆ 
<script>
// String
let name = "สมชาย";
// Number
let score = 90;
// Boolean
let pass = true;
// Array
let subjects = ["Math", "JS", "CSS"];
// Object
const person = { name: "สมชาย", age: 20 };

console.log(name, score, pass, subjects, person);
</script>
* เปิด Dev tools (F12) จะเห็นค่าต่างๆ

Function (ฟังก์ชัน) ใน JavaScript

ฟังก์ชัน คือ คำสั่งกลุ่มหนึ่งที่เรียกใช้ซ้ำได้
ตัวอย่าง: 
<script>
function sayHello(name) {
  console.log("สวัสดี", name);
}

sayHello("สมชาย");
</script>
* ดูคำว่า "สวัสดี สมชาย" ใน console

if, else, while, do while, for

if / else เงื่อนไขใน JS: 
<script>
let score = 60;

if (score >= 50) {
  console.log("ผ่าน");
} else {
  console.log("ไม่ผ่าน");
}
</script>
while, do while, for (ลูปวนซ้ำ)
  • while: วนจนเงื่อนไขเป็นเท็จ
  • do while: ทำก่อน 1 ครั้งเสมอ
  • for: นิยมใช้เมื่อรู้จำนวนรอบ
<script>
// while
let n = 0;
while(n < 3) {
  console.log("while", n);
  n++;
}

// do while
let m = 0;
do {
  console.log("do while", m);
  m++;
} while(m < 2);

// for
for(let i=0; i<4; i++) {
  console.log("for", i);
}
</script>
* ดูลำดับใน console
Tip: สามารถผสม if, ลูป, ฟังก์ชันในการเขียนโปรแกรมแก้ปัญหาได้!
ลองฝึก JS:
  1. สร้างตัวแปร array แล้วใช้ for วนแสดงทีละค่าใน console
  2. สร้างฟังก์ชันเช็คเลขคู่/คี่ แบบใช้ if
  3. เขียนลูป while ให้แสดงเลข 1-5

Roadmap JS มือใหม่

  • 1. เข้าใจชนิดข้อมูล ตัวแปร (string, number, boolean, array, object)
  • 2. ใช้ if/else, while, do while, for
  • 3. สร้าง function เรียกใช้งานซ้ำ
  • 4. นำไปใช้ในโค้ดจริง เช่น ตรวจสอบ/วนแสดงข้อมูล
ทิปส์: ทดลองเปลี่ยนค่าตัวแปร หรือลูป แล้วดูผลใน console จะช่วยให้เข้าใจดีขึ้น!
