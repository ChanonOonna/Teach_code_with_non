บทที่ 2 HTML Document Structure (โครงสร้างไฟล์ที่ต้องรู้)

สิ่งสำคัญที่ครอบคลุมโครงสร้างหลัก

  • <!DOCTYPE html>: ไม่ใช่แท็ก HTML แต่เป็นคำสั่งบอกเบราว์เซอร์ว่า "นี่คือเอกสารมาตรฐาน HTML5 ตัวล่าสุดนะ" เพื่อป้องกันไม่ให้เบราว์เซอร์เข้าสู่โหมดประมวลผลเพี้ยนแบบยุคโบราณ (Quirks Mode)
  • <html>: Root element หรือเปลือกนอกสุดที่คุมรหัสโค้ดทั้งหมด นิยมใส่แอตทริบิวต์ภาษาเช่น lang="th" เพื่อส่งสัญญาณให้โปรแกรมแปลภาษาและโปรแกรมช่วยอ่านหน้าจอมือถือรับทราบ

ส่วนข้อมูลหลังบ้าน และส่วนแสดงผล

  • <head>: เป็นที่อยู่อาศัยของข้อมูลทางเทคนิคที่ ไม่ปรากฏกายบนพื้นที่หน้าเว็บหลัก เช่น การระบุชุดตัวอักษร <meta charset="UTF-8">, การตั้งชื่อแถบแท็บบนเบราว์เซอร์ <title> และไฟล์ CSS ที่จะนำมาผูก
  • <body>: กล่องเก็บวัตถุสิ่งของทุกชิ้นที่จะไปโผล่แสดงผลจริงแก่สายตาผู้ใช้งาน ไม่ว่าจะเป็นปุ่มกด ข้อความ ตาราง รูปภาพ หรือฟอร์มสมัครสมาชิก
